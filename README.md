# machine-learning-diabetes-study
Estudo prático de Machine Learning desenvolvido durante curso de Inteligência Artificial, utilizando Python, Pandas e Scikit-Learn para classificação de dados.

Projeto de Classificação de Diabetes com Machine Learning
Sobre o projeto
Este projeto foi desenvolvido como parte de um estudo prático durante um curso de Inteligência Artificial e Ciência de Dados.
O objetivo foi compreender as principais etapas de um projeto de Machine Learning, desde a análise inicial dos dados até o treinamento e avaliação de um modelo de classificação.
Ferramentas utilizadas
Python
Pandas
NumPy
Matplotlib
Scikit-Learn
Etapas realizadas
Importação e análise dos dados
Estatísticas descritivas
Visualização de distribuições
Separação entre dados de treino e teste
Padronização dos dados com StandardScaler
Treinamento do modelo K-Nearest Neighbors (KNN)
Geração de previsões
Avaliação por matriz de confusão e acurácia
Resultados
O modelo obteve aproximadamente 76% de acurácia na base de testes.
Durante o desenvolvimento do projeto também foi analisada a limitação da acurácia como métrica única de avaliação, especialmente em bases de dados com distribuição desigual entre classes.
Objetivo de aprendizado
Este projeto teve como foco o aprendizado dos conceitos fundamentais de Machine Learning e do fluxo de desenvolvimento de modelos preditivos utilizando a biblioteca Scikit-Learn.
